# lmdmap

## Installation

```{sh}
pip install git+https://github.com/3d-omics/lmdmap.git
```

## Update

```{sh}
pip uninstall lmdmap
pip install git+https://github.com/3d-omics/lmdmap.git

```


## Usage

```{sh}
lmdmap G007bI105A input/241113G007bI105post.jpg
lmdmap G007bI105A input/241113G007bI105post.jpg --draw-microsamples
```
